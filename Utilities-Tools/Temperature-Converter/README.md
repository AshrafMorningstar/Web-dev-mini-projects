# Temperature Converter

## Description
Quick and easy temperature converter that instantly converts between Celsius, Fahrenheit, and Kelvin.

## Features
- Real-time bidirectional conversion
- Convert between Celsius, Fahrenheit, Kelvin
- All three units visible simultaneously
- Visual temperature scale
- Preset common temperatures
- Responsive design
- Copy to clipboard functionality

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Use
1. Enter temperature in any field
2. Other fields auto-update
3. Or click preset buttons for common temperatures
4. Copy result to clipboard

## Conversion Formulas
- Celsius to Fahrenheit: (C × 9/5) + 32
- Celsius to Kelvin: C + 273.15
- Fahrenheit to Celsius: (F - 32) × 5/9
- Fahrenheit to Kelvin: (F - 32) × 5/9 + 273.15
- Kelvin to Celsius: K - 273.15
- Kelvin to Fahrenheit: (K - 273.15) × 9/5 + 32

## Installation
1. Open index.html in your browser
2. Start converting!
