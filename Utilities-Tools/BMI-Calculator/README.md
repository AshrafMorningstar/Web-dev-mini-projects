# BMI Calculator

## Description
A simple and effective BMI (Body Mass Index) calculator that helps you determine your body mass index and health category.

## Features
- Input height in cm or feet/inches
- Input weight in kg or lbs
- Real-time BMI calculation
- Health category display (Underweight, Normal, Overweight, Obese)
- Color-coded results
- Unit conversion support
- Calculation history (localStorage)
- Responsive design

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Use
1. Enter your height
2. Enter your weight
3. Select units (metric or imperial)
4. Click Calculate
5. See your BMI and health category
