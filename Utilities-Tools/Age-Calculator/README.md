# Age Calculator

## Description
Instant age calculator that calculates your exact age in years, months, days!

## Features
- Calculate age from birth date
- Display in years, months, days format
- Next birthday countdown
- Zodiac sign detection
- Responsive design

## Technologies
- HTML5, CSS3, Vanilla JavaScript
