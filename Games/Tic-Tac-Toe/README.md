# Tic-Tac-Toe Game

## Description
Classic Tic-Tac-Toe game with AI opponent. Play on a 3x3 board and try to get three in a row!

## Features
- Play as X against AI opponent
- AI uses minimax algorithm (Hard mode)
- Score tracking
- Easy and Hard difficulty modes
- Win/Loss/Draw detection
- Reset game button
- Responsive design
- Smooth animations

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## How to Play
1. Click a square to make your move (you are X)
2. AI will automatically make its move (O)
3. First to get 3 in a row wins!
4. Select difficulty level for different AI strategies

## Installation
1. Open index.html in your browser
2. Select difficulty level
3. Start playing!

## Difficulty Levels
- **Easy:** AI makes random moves
- **Hard:** AI uses minimax algorithm for optimal play
