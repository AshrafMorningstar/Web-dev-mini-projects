# 2048 Game

## Description
A clone of the popular 2048 sliding tile puzzle game. Combine tiles with the same number to reach 2048!

## Features
- 4x4 game grid
- Smooth tile animations
- Score tracking
- High score saved to localStorage
- Keyboard controls (Arrow keys)
- Mobile touch support
- Game over detection
- New game button

## Technologies Used
- HTML5
- CSS3 with animations
- Vanilla JavaScript (ES6+)

## How to Play
1. Use arrow keys to move tiles
2. When two tiles with the same number touch, they merge
3. Keep combining tiles to reach 2048
4. Game ends when no more moves possible
