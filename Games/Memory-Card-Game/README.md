# Memory Card Game

## Description
Classic Memory/Concentration Card Matching Game. Test your memory by finding matching pairs of cards!

## Features
- 16 cards in a 4x4 grid
- Click to flip and match pairs
- Score tracking (attempts, matches)
- Timer (optional)
- Difficulty levels (Easy 4x4, Medium 4x6, Hard 6x6)
- Best time saved
- Sound effects
- Responsive touch support
- Reset game button
- Victory screen

## Technologies Used
- HTML5
- CSS3 with animations
- Vanilla JavaScript (ES6+)
- LocalStorage for persistence

## How to Play
1. Click on cards to reveal
2. Try to find matching pairs
3. Match all pairs to win
4. Fastest time = best score

## Difficulty Levels
- **Easy:** 8 pairs (4x4 grid)
- **Medium:** 12 pairs (4x6 grid)
- **Hard:** 18 pairs (6x6 grid)

## Installation
1. Open index.html in browser
2. Select difficulty
3. Start playing!
