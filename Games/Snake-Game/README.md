# Snake Game

## Description
Classic Snake game built with HTML5 Canvas. Navigate your snake to eat food and grow without hitting walls or yourself!

## Features
- Classic Snake gameplay
- Score tracking
- High score persistence
- Difficulty levels (Slow, Normal, Fast, Ultra)
- Smooth animations
- Keyboard controls (Arrow keys or WASD)
- Mobile touch controls
- Pause/Resume game
- Sound effects
- Game over screen
- Touch swipe support

## Technologies Used
- HTML5 Canvas
- Vanilla JavaScript (ES6+)
- LocalStorage

## How to Play
1. Use arrow keys or WASD to move
2. Eat red food to grow and score points
3. Avoid hitting walls and yourself
4. Game speeds up as you progress

## Difficulty Levels
- **Slow:** 80ms per frame
- **Normal:** 100ms per frame
- **Fast:** 60ms per frame
- **Ultra:** 40ms per frame

## Installation
1. Open index.html in browser
2. Select difficulty
3. Start playing!

## Controls
- Arrow Keys or WASD to move
- SPACE to pause/resume
- R to restart
