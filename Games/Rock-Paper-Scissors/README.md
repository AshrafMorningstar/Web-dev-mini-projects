# Rock Paper Scissors

## Description
Classic Rock-Paper-Scissors game where you play against an AI opponent. Track your wins, losses, and draws.

## Features
- Play against AI opponent
- Score tracking (Wins/Losses/Draws)
- Animated button interactions
- Result display with emojis
- Statistics saved to localStorage
- Responsive design
- Best of series option (Best of 3, 5, 7)

## Technologies Used
- HTML5
- CSS3 with animations
- Vanilla JavaScript

## How to Play
1. Click Rock, Paper, or Scissors
2. AI makes its choice
3. Compare to determine winner
4. First to win required matches wins the game

## Game Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## Installation
1. Open index.html in your browser
2. Start playing!
